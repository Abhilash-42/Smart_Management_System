# Smart Management System

Smart Management System is a web-based application developed to manage student records, attendance, and performance analytics efficiently.

## Features

- Student Management
- Attendance Tracking
- Performance Analytics
- Dashboard Statistics
- Admin & Viewer Login
- PDF Report Generation
- Excel Import & Export
- Responsive User Interface

## Technologies Used

- Next.js
- React.js
- MongoDB Atlas
- Mongoose
- NextAuth
- Tailwind CSS
- Chart.js

## Installation

1. Clone the repository

```bash
git clone https://github.com/Abhilash-42/Smart_Management_System.git
```

2. Navigate to the project folder

```bash
cd Smart_Management_System
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env.local` file and add:

```env
MONGODB_URI=your_mongodb_connection_string
NEXTAUTH_SECRET=your_secret_key
NEXTAUTH_URL=http://localhost:3000
```

5. Run the application

```bash
npm run dev
```

6. Open in browser

```text
http://localhost:3000
```
